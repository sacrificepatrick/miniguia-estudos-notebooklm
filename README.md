# Miniguia de Estudos - NotebookLM

## Contexto e Objetivos
Este repositório foi criado como parte do desafio da DIO **"Treinando uma IA de Aprendizagem: Explore o Poder do NotebookLM"**.  
O tema escolhido foi **Automação de Dados com IA**.  
O objetivo é compreender como a Inteligência Artificial pode apoiar processos de automação, organizar conhecimento e criar um guia de estudo para revisões futuras.

## Curadoria de Fontes
As principais fontes utilizadas para alimentar o NotebookLM foram:
- [Documentação da OpenAI sobre modelos de linguagem](https://platform.openai.com/docs)  
- [Artigo introdutório sobre Machine Learning - Towards Data Science](https://towardsdatascience.com/)  
- [Guia de Pandas para manipulação de dados](https://pandas.pydata.org/docs/)  

## Engenharia de Prompts e Cicatrizes
Exemplos de prompts utilizados:
- "Explique como a IA pode automatizar rotinas de análise de dados."  
- "Quais são os riscos de alucinações em modelos de linguagem?"  
- "Como a janela de contexto influencia a qualidade das respostas?"  

**Resultados e dificuldades encontradas:**
- Algumas respostas vieram vagas → refinei os prompts adicionando contexto.  
- A IA trouxe boas referências, mas precisei validar com as fontes originais.  
- Aprendi que prompts claros e específicos geram melhores resultados.

## Miniguia de Estudo (Entrega Final)
### Resumos Estruturados
- **Automação de Dados:** uso de scripts e IA para reduzir tarefas repetitivas.  
- **Tokenização:** processo de dividir texto em unidades menores (tokens).  
- **Janela de Contexto:** limite de informações que o modelo consegue considerar.  

### Glossário
- **Prompt Engineering:** técnica de elaborar instruções claras para IA.  
- **Alucinação:** quando a IA gera informações incorretas ou inventadas.  
- **SQL:** linguagem para manipulação de bancos de dados relacionais.  

### Prompts Reutilizáveis
- "Resuma os principais conceitos de automação de dados."  
- "Liste vantagens e desvantagens do uso de IA em rotinas empresariais."  
- "Explique como aplicar Pandas em um projeto prático."
